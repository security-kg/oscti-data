Fantom ransomware impersonates Windows update

Reading Time: ~ 3 min.

Fantom behind the scenes

In an attempt to conceal malicious intention, the authors of this ransomware modified the file properties to show copyright and legal trademarks mimicking a Windows update.

Once this dropper is executed, the payload ‚ÄúWindowsUpdate.exe‚Äù is dropped in AppData\Local\Temp displaying the fake Windows Update screen as shown below. DECRYPT_YOUR_FILES.HTML ransom note

Encryption is done using AES-128 encryption and when a file is encrypted it will append ‚Äú.fantom‚Äù to the extension of the file. Also in every directory that a file is encrypted, a standard ransom note ‚ÄúDECRYPT_YOUR_FILES.HTML‚Äù is created.
